Project Description

Virtuosic Media is my web development and hosting company. I wanted this site to be modern, responsive, and blazingly fast, so I eschewed a content management system and just coded it by hand in php, html5, css3, and just a bit of jQuery to handle the navigation transitions, lightboxes, and form verification.

Features

  • Single page layout
  • Paralax background images
  • Smooth Scrolling navigation
  • Responsive grid-based gallery
  • Lightbox pop-ups for individual images and files
