Project Name
Eagle Y Condor

Site Address

https://www.eagleycondor.org

Project Description

Eagle Y Condor is a humanitarian and animal relief organization that provides medical care and education to the animals and children of Ecuador. As such, their logo and color scheme features the national colors of Ecuador. The site is very image intensive, yet still scored an 88 performance grade on Pingdom.

Features

  • Custom Responsive Grid
  • Sticky menu
  • Responsive images
  • Donation and PayPal integration
  • Five custom content types
  • Form Integration
  • Image Focal Point feature for auto focusing thumbnails
